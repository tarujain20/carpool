# Carpool

This project will help the community to find carpools to work easily!

Its deployed on heroku (simple version). I am still working on this slowly but steadly. Check it out:
http://still-ocean-8434.herokuapp.com/

### Prerequisites

git


### Run the specs

The simplest way to run all the specs:

```
rake (from the root dir)
```

### Run the Application

 The simplest way to start this server is:

```
rails s
```

Now browse to the app at `http://localhost:3000/`.
